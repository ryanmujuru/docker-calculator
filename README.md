Dockerized Python Calculator

Description

This project is a simple Python calculator that has been containerized using Docker.

The calculator performs:

* Addition
* Subtraction
* Multiplication
* Division

Docker Build

The Docker image was built using:

```bash
docker build -t python-calculator .
```

Run the Container

The application was run using:

```bash
docker run -it --rm python-calculator
```

Technologies Used

* Python
* Docker
* Docker Containers

Skills Demonstrated

* Creating a Dockerfile
* Building a Docker image
* Running a Docker container
* Containerizing an application
